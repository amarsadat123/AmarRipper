# 🚀 AmarRipper

**AmarRipper** is a Python-based network stress testing tool developed for educational purposes and security auditing. It utilizes multi-threading to simulate high-volume HTTP traffic, helping administrators test the resilience of their infrastructure against resource exhaustion.

---

## 🛠️ Features
* **Multi-threaded Engine:** High-performance packet generation using Python's `threading` library.
* **User-Agent Randomization:** Rotates headers to simulate different devices and bypass simple filtering.
* **Cross-Platform:** Works on Windows, Linux (Kali), and macOS.
* **Interactive CLI:** Easy-to-use interface with a custom ASCII banner.
